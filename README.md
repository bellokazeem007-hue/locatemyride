## About

LocateMyRide247 is a sleek, fully responsive landing page for a vehicle tracking and recovery service. Designed to build trust and drive conversions, it showcases:

- **Real-time GPS tracking** simulation with a moving vehicle on a map.
- **Transparent pricing** with three tiers (Basic, Premium, Pro, Platinum).
- **Animated stats** that count up on load (5,000+ customers, 10K+ vehicles tracked, 24/7 support).
- **Modern dark theme** with neon green accents for a high‑tech feel.
- **Fully interactive** – FAQ accordion, sign‑up modal, mobile‑friendly navigation.

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, flexbox/grid)
- Vanilla JavaScript (intersection observer, counter animation, event handling)
- Font Awesome 6 for icons
- Google Fonts (Inter)
